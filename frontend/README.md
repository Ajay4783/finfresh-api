# FinFresh Frontend

## Component Structure
- **Pages:** Login, Register, Dashboard.
- **State Management:** React `useState` and `useEffect` for local state handling.
- **API Layer:** Centralized API calls using Axios with JWT Bearer tokens in headers.

## Functional Requirements
- Handles Loading, Error, and Empty states.
- Auto-redirects to login if token is missing.
- Dynamic color coding for Income (Green) and Expense (Red).
